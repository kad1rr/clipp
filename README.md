# Clipp Command Line Interface (Clipp CLI)

Clipp CLI is a command-line tool designed to help you quickly save and retrieve code snippets that you use frequently in your projects. It simplifies the process of managing and reusing code blocks, making your coding workflow more efficient.

## Installation

Ensure you have [Rust](https://www.rust-lang.org) installed.

## Usage

```bash
clipp (filename)
clipp save (filename)
clipp [args]
```

## Commands
```bash
clipp (filename)
clipp save (filename)
```

### Examples
Save a code snippet:

```bash
clipp save my_function.py
```
Display the content of a saved code snippet:

```bash
clipp my_function.py
```
